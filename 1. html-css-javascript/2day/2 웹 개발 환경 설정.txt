========================
      웹 브라우저와 웹 편집기
========================
웹 브라우저: 사용자가 인터넷에 접속하여 웹 페이지를 탐색하고 표시하는 
                응용 소프트웨어이다. 사용자가 입력한 URL을 통해 웹 서버에 요청을
                보내고, 서버로부터 받은 HTML,CSS, JavaScript 등의 파일을 해석하여
                화면에 표시한다
                ex ) Chrome, Firefox, Safari, Edge 등
  * 주요 역할 및 기능
    - 웹 페이지의 랜더링: HTML, CSS, JavaScript를 해석하여 시각적으로 표현한다
    - 통신 프로토콜 지원: HTTP/HTTPS 포로토콜을 사용하여 서버와 데이터를 
                               주고받음
    - 웹 표준 지원: 최신 HTML5, CSS3, JavaScript 기능을 지원하여
                       다양한 웹페이지를 올바르게 렌더링한다
    - 쿠기와 캐시 관리: 쿠키와 캐시를 저장하여 사용자 정보를 관리하고,
                            웹페이지 로딩을 최적화한다
    - 개발자 도구 제공: 대부분의 웹 브라우저는 개발자 도구를 제공하여,
                            웹 개발자가 HTML 구조, CSS 스타일, JavaScript 동작 등을
                            확인하고 디버깅할 수 있다
  * 웹 브라우저 중요성
    - 사용자와 웹 애플리케이션 사이의 인터페이스 역할을 함
    - 개발자에게 개발자 도구(F12)를 제공하여, 실시간 디커깅과 퍼포먼스 최적화에
      도움을 줌
          

웹 편집기: 웹 페이지를 작성하고 편집하기 위한 도구로, HTML, CSS, JavaScript 등의
             코드를 작성하고 수정하는 데 사용된다. 웹 편집기는 코드 작성의 효율성을
             높이고, 실시간 미리보기 기능 등을 통해 개발 과정을 지원한다
  * 종류
    - 텍스트 기반 편집기: 단순 텍스트 편집기로, 코드 작성과 편집을 위한 
                               최소한의 기능을 제공한다. ex) 메모장, Vim, Emacs 등
    - 전용 웹 편집기: 웹 개발에 특화된 기능을 제공하며, 코드 자동 완성, 문법 강조,
                          실시간 미리보기 등을 지원한다. 
                          ex) 어도비 드림워버(Adobe Dreamweaver), 나모 웹에디터
    - 통합 개발 환경(IDE): 코드 작성, 디버깅, 배포 등 개발의 모든 과정을 지원하는
                               종합적인 도구이다. 
                               ex) 비주얼 스튜디오 코드(Visual Studio Code),
                                   이클립스(Eclipse), 인텔리제이(IntelliJ)
  * 주요 역할 및 기능
    - 코드 작성 및 편집: HTML, CSS, javaScript 등의 웹 언어로 웹페이지를 작성하고
                             수정할 수 있다
    - 문법 강조와 자동 완성: 코드를 쉽게 읽을 수 있도록 문법을 강조하고,
                                   자동 완성 기능을 제공하여 코딩 효율성을 높인다
    - 파일 관리: 프로젝트 폴더 구조를 관리하고, 각 파일을 쉽게 열고 편집할 수 있다
    - 버전 관리 지원: Git과 통합되어 프로젝트의 변경 이력을 관리하고,
                          여러 명이 협업을 할 수 있다
    - 실시간 미리보기: 작성한 코드를 즉시 브라우저에게 미리보기로 확인할 수 있다
    - 확장성: 다양한 플러그인과 확장을 통해 필요에 따라 기능을 추가할 수 있다
  * 웹 편집기의 중요성
    - 코드 작성과 편집을 쉽게 하고, 개발자가 코드 품질을 높일 수 있도록 돕는다
    - 디버깅과 협업 기능을 통해 개발 생산성을 높이는 데 중요한 역할을 한다

웹 브라우저와 웹 편집기의 차이점
  * 웹 브라우저는 웹페이지 표시 및 상호작용을 제공하지만,
    웹 편집기는 웹페이지 코딩 및 편집하는 역할이다
  * 웹 브라우저는 HTML,CSS, JavaScript 랜더링을 하지만,
    웹 편집기는 코드 작성, 자동 완성, 디버깅을 하는 기능이 있다
  * 웹 브라우저는 최종 사용자 및 웹 개발자가 쓰지만,
    웹 편집자는 주로 웹 개발자만 쓴다

========================
        웹 편집기 환경 설정
========================
VS Code 설치
  1. VS Code의 웹 사이트(code.visualstudio.com)에 접속
  2. [Download for Windows(Mac)] 버튼 클릭
  3. 다운로드 창에서 다운로드한 파일 클릭 > 동의합니다 > 
  4. 추가 작업 전부 체크 해주는 것을 추천 > 설치

한국어로 설정
  1. VS Code 실행 > 왼쪽 사이드바 확장(Extensions) 아이콘 클릭
  2. 검색창에 korean 입력 >
  3. 'Korean Language Pack for Visual Studio Code'를 찾아 설치
  4. 설치 후 VS Code 재시작
  * 또다른 방법
    1. 단축기(Ctrl + Shift + P)를 누르기 > 커맨드 팔레트 실행 후
    2. 커맨드 팔레트 실행 후 > Configure Display Langage 검색 > [한국어] 선택
    3. 설치 후 VS Code 재시작

색 테마 변경
  1. 왼쪽 사이드바 관리 아이콘 클릭 > 테마 > 색 테마 > 원하는 테마 적용

폰트 크기 조절
  1. 왼쪽 사이드바 관리 아이콘 클릭 > 설정(Ctrl + ,) > 
  2. 검색 창 font size 입력 > 원하는 폰트 크기 설정

탭 크기 조절
  1. 왼쪽 사이드바 관리 아이콘 클릭 > 설정(Ctrl + ,) >
  2. 검색 창 Tab Size 입력 > 원하는 탭 크기 설정(2 권장)

자동 저장 설정
  1. 오른쪽 메뉴에서 파일 클릭 > 자동 저장 찾아 활성화

필수적으로 필요한 확장 프로그램
  * Korean Language Pack for Visual Studio Code: 한국어 언어로 변환
  * Prettier - Code formatter: 코드의 스타일을 유지하기 위해 자동으로 
                                     포맷팅해주는 도구
  * Material Theme: 다양한 데마를 제공하여 코드 편집기의 시각적 환경을 개선함
  * Material Icon Theme: 파일 및 폴더 아이콘을 시각적으로 구분하여
                               파일 탐색에 용이함
  * Bracket Pair Colorizer 2: 중첩된 괄호의 생상을 구분해주는 도구
  * Rainbow CSV: 파일의 각 열을 서로 다른 색상으로 강조하여 지원하는 도구
  * indent-rainbow: 들여쓰기 레벨에 따라 색상을 부여하여 코드 구조를 
                         쉽게 파악할 수 있음
  * Live Server: 로컬 개발 서버를 실행하여 코드 변경 시 브라우저에서
                   실시간으로 결과를 확인할 수 있음
  * GitLens: Git의 히스토리와 변경 사항을 시각적으로 표시하여 버전 관리에 용이함

언어별 확장 프로그램
  * Python: Microsoft에서 제공하는 공식 Python 확장 프로그램으로, 
              코드 자동 완성, 디버깅, 테스트, 가상 환경 관리 등 다양한 기능을 지원함
  * Jupyter: Jupyter 노트북을 VS Code 내에서 실행하고 편집할 수 있도록 지원함
  * C/C++: Microsoft에서 제공하는 공식 확장 프로그램으로, 
              코드 자동 완성, 디버깅, 코드 분석 등 다양한 기능을 지원합니다.

  * C/C++ Extension Pack: C/C++ 개발에 필요한 도구들을 모아놓은 확장 팩으로, 
                                 코드 포맷터, 디버거, 코드 스니펫 등을 포함하여 
                                 생산성을 높여줍니다.
  * C#: Microsoft에서 제공하는 공식 C# 확장 프로그램으로, 
         코드 편집, 디버깅, IntelliSense 등을 지원함
  * ASP.NET Core Snippets: ASP.NET Core 개발 시 자주 사용하는
                                  코드 스니펫을 제공함
  * Language Support for Java™ by Red Hat: Red Hat에서 제공하는 
                 공식 Java 확장 프로그램으로, 코드 자동 완성(IntelliSense), 
                 디버깅, 코드 분석 등 다양한 기능을 지원
  * Java Extension Pack: Java 개발에 필요한 다양한 도구와 라이브러리를 
                              한 번에 설치할 수 있는 확장 팩
  * Spring Boot Extension Pack: Spring Boot 개발을 위한 도구 모음으로, 
                                        프로젝트 생성, 실행, 디버깅 등을 지원함

프론트엔드 확장 프로그램
  * Emmet: HTML과 CSS 코드를 빠르게 작성할 수 있도록 도와주는 도구로, 
              간단한 약어를 입력하면 자동으로 코드 블록을 확장해줌
  * CSS Peek: HTML 파일에서 클래스나 ID를 클릭하면 해당 스타일 정의로 
                 바로 이동하거나 미리보기를 제공하여, 스타일 시트를 
                 쉽게 탐색할 수 있게 해줌
  * IntelliSense for CSS class names in HTML: HTML 파일에서 CSS 클래스 이름을 
                   입력할 때, 현재 프로젝트에 정의된 클래스 이름을 자동 완성으로 
                   제안해줌
  * HTML CSS Support: HTML 파일 내에서 CSS 코드에 대한 IntelliSense를 제공하여,
                              CSS 속성 및 값에 대한 자동 완성과 유효성 검사를 지원함
  * JavaScript (ES6) code snippets: JavaScript ES6 문법에 대한 코드 스니펫을 
                 제공하여, 자주 사용하는 코드 패턴을 빠르게 입력할 수 있도록 도와줌
  * ESLint: JavaScript 코드의 문법 오류와 스타일 문제를 감지하고 수정할 수 있도록              도와주는 린터 도구로, 코드 품질을 유지하는 데 유용함
  * Auto Close Tag & Auto Rename Tag: HTML 태그를 작성할 때 자동으로 닫는 \                     태그를 생성해 주고, 태그 이름을 변경할 때 시작 태그와 종료 태그를                     동시에 수정해 줌

========================
       간단한 웹 문서 만들기
========================
작업 폴더 추가하기
  1. 파일 탐색기 열기 > 작업하고 싶은 저장소(C: 혹은 D:)선택 >
  2. 오른쪽 위 사이드바에 새 폴더(Ctrl + Shift + N) 아이콘 클릭 > 이름 설정
  3. VS 코드 실행 > 오른쪽 메뉴에서 파일 선택 > 폴더 열기 > 
  4. 2번에서 만든 폴더 클릭 > 폴더 선택

작업 폴더에 새 파일 만들기
  1. 왼쪽 탐색 창에서 폴더를 선택 > 폴더 목록 위에 있는 새파일 선택 > 
  2. 이름 설정(index.html) > 엔터(Enter)

코드 입력하기

  "웹 개발 기초"
  "HTML" 
  "CSS"        
  "자바스크립트"

  * 코드 입력 후 저장
    - 저장 단축키: Ctrl + S
  * 'HTML' 이라고 입력하고 Enter를 눌렸을 때 <HTML></HTML>이 입력될 수도 있다
    이럴 경우 자동으로 입력된 내용을 삭제하고 HTML 다음에 Space를 눌러 한칸 띄고
    Enter를 누른다

작성 결과 확인하기
  * Live Server라는 확장 프로그램 설치히고 오기
  * VS Code에서 작성하고 아래 사이드바에서 Go Live 클릭

========================
               마크업하기
========================
마크업: 웹 문서의 각 부분이 어떻게 표시되어야 하는지 내용마다 코드를 붙이는 것

"<h1>웹 개발 기초</h1>
<p>HTML</p>
<p>CSS</p>
<p>자바스크립트</p>"

<h1> 태그: 제목
<p> 텍스트 단락
  * <h1> 태그를 입력하면 자동으로 <h1></h1>으로 수정될 것이다.(<p>도 포함)
    <h1>과 <h1> 태그 사이에 제목이 오도록 입력한다
  * Ctrl + S을 눌러 문서 저장은 필수적으로 해주자
    (저장이 안되어 있을 시 검정 점이 뜸)

실습 결과 확인
  * 열려 있는 브라우저 창에서 새로고침 한다
  * '웹 개발 기초'는 제목으로 커졌고 나머지 내용은 한 줄 씩 나타난다

마크업한다: 이렇게 웹 문서 내용마다 태그를 붙여서 각각 역할을 결정 짓는 것

========================
             웹 개발자 도구
========================
웹 개발자 도구 창 열기
  1. Live Server(Go Live)을 눌려 브라우저 열기
  2. Ctrl + Shift + l(혹은 F12)를 누르면 웹 개발자 도구 창이 열린다
    (브라우저 창의 빈 공간을 마우스 오른쪽 버튼으로 누른 후 [검사]를 선택해도 됨)

웹 개발자 도구 한글 표시하기
  1. 웹 개발자 도구 창의 오른쪽 위에 있는 '설정' 을 클릭
  2. [환경설정]의 [언어] 항목을 [한국어]로 지정
  3. 한글로 바꾼 후 x를 클릭해서 닫기

HTML 코드 살펴보기
  * 웹 개발자 도구 차엥서 [요소] 탭을 클릭하면 화면이 왼쪽과 오른쪽 2개로 나뉨
  * 왼쪽에는 HTML 코드가 나타나고, 오른쪽에는 CSS 속성이 보인다
  * HTML 태그 위로 마우스 포인터를 가져가면 그 태그에 웹 브라우저에서 어떻게
    적용되는지 화면에 표시해준다
    ex) 웹 개발자 도구 창에서 <h1>웹 개발 기초</h1> 코드 위로 마우스 커서를
         가져가면 그 태그가 적용된 제목이 있는 줄 전체에 배경색을 추가해서 알려줌

코드 접기 및 펼치기
  * 웹 문서를 이루는 코드는 꽤나 길어서 [요소] 탭에서 모든 코드가 한 번에 나타나지
    않고 일부 코드가 접혀 있어서 보이지 않는다
  1. 접힌 코드 앞에는 ▶ 표시가 있다. <head> 태그 앞에 ▶을 클릭
  2. ▶가 ▼로 바뀌면서 <head> 태그와 </head> 태그 사이에 있던 코드가 펼쳐짐

웹 개발자 도구 창의 위치 옮기기
  1. 개발자 도구 창 오른쪽 위에 있는 점 3개를 클릭하면 개발자 도구창의 위치를
     선택하는 아이콘 4개가 나타난다
  2. 왼쪽부터 순서대로 별도 창으로 분리, 왼쪽 고정, 아래 고정, 오른쪽 고정을 뜻한다
  3. 이중에서 원하는 위치를 선택하여 개발자 도구 창의 위치 바꾸기
